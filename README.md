# Daily-Task-Manager

This is a daily task manager application project created in flutter.


## Getting Started
Go to project root and execute the following command in console to get the required dependencies:

    flutter pub get 

## Required Tools
- Android Studio
- Visual Studio Code (Optional)
- Flutter
- Dart
